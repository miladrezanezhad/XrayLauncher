# Changelog

All notable changes to this project will be documented in this file.  
This project adheres to [Semantic Versioning](https://semver.org/).

---

## [v1.0.0] - 2026-01-03
### Added
- First public release of **Xray Launcher**
- Embedded **Xray-core v25.12.8** inside the app
- Built on **.NET Framework 4.8**
- Config list management with persistent storage (`configs.txt`)
- One-click proxy activation/deactivation via Windows registry
- Tray icon with quick actions (Start, Enable Proxy, Disable Proxy, Exit)
- Modern UI: borderless form, rounded corners, drag support
- Display of public IP, connection status, and simulated speed
- Automatic cleanup of temporary files and full internet restore on exit
